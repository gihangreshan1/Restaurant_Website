## 📖 Project Overview

The **Restaurant Website** is designed to provide customers with essential information about a restaurant in a clean, visually appealing, and user-friendly format.

The website includes:
- Restaurant introduction
- Menu and services
- Contact details
- Responsive design for desktop and mobile users

This project was developed as part of a university **Software Engineering** course, with focus on:
- Requirement analysis
- UI/UX design
- Front-end implementation
- Team collaboration
- Documentation and testing

---

## ✨ Features

- ✅ Responsive homepage
- ✅ Navigation bar with smooth section access
- ✅ About Us section
- ✅ Menu section
- ✅ Services section
- ✅ Contact section
- ✅ Image gallery
- ✅ Mobile-friendly layout
- ✅ Interactive UI elements with JavaScript

## 🚀 Getting Started

### 1) Clone the repository

```bash
git clone https://github.com/gihangreshan1/Restaurant_Website.git
```

### 2) Navigate into the project directory

```bash
cd Restaurant_Website
```

### 3) Run the project

Open `index.html` in your preferred web browser.

No additional dependencies or installation steps are required.
